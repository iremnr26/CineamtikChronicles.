About Us

This site is a blog site generally related to movies, and the user is asked to log in on the first page. If the user does not have an account, he/she can log in after registering on the site. Afterwards, the user can generally list three different contents. These are the lists of all movies, the top 20 movies, and their own movie list. The user can add movies to his/her own list from the list that includes all movies and the top 20 movies. Users can also search for the name of the movie on these pages. The user can remove the movie from his own movie list or change the rating of the movie.
